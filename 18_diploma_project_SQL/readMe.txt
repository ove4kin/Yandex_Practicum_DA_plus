SQL-запросы к базе данных сервиса по продаже книг. Часть 4 дипломного проекта.

Задача:
Проанализировать базу данных для формирования ценностного предложения для нового продукта: запросы к базе данных 
SQL с помощью Python

Используемый стек проекта: Python, Pandas, SQLAlchemy, PostgreSQL

Результат:

Работа основана на базе данных книжного магазина и информации о книгах, издательствах, авторах и пользовательских обзорах книг. 
Было создано удаленное соединение к базе данных в Python, написаны запросы в соответствии с задачами.

Цели проекта выполнены.

Статус проекта: Завершён
