Описание проектов этого репозитория:

********************************************************

01_basic_python — проект «Музыка больших городов».


На реальных данных Яндекс.Музыки c помощью библиотеки Pandas и её возможностей проверяем данные и сравниваем поведение и предпочтения 
пользователей двух столиц — Москвы и Санкт-Петербурга.


Используемый стек проекта: Python, pandas


********************************************************

02_data_preprocessing — проект ««Исследование надёжности заёмщиков»

На основе статистики о платёжеспособности клиентов исследуем влияет ли семейное положение и количество детей клиента на факт 
возврата кредита в срок.

Используемый стек проекта: Python, pandas


********************************************************

03_research_analysis — проект «Исследование объявлений о продаже квартир»

Используя данные сервиса Яндекс.Недвижимость, определяем рыночную стоимость объектов недвижимости и типичные параметры квартир.

Используемый стек проекта: Python, pandas


********************************************************

04_prefabricated_project_1 — проект «Выявление текущих трендов российского кинопроката»

Изучаем текущие тренды на рынке российского кинопроката с целью выявления интересных закономерностей у картин, 
имеющих государственную поддержку.

Используемый стек проекта: Python, Pandas, Numpy, Matplotlib, Seaborn


********************************************************

05_basic_sql — решение задач с применением базовых элементов языка SQL

Произвести различные выгрузки данных авиакомпаний с помощью SQL.

Используемый стек проекта: SQL


********************************************************

06_how_to_tell_story_with_data — проект «Рынок заведений общественного питания Москвы»


Исследование рынка общественного питания на основе открытых данных, подготовка презентации для инвесторов.

Используемый стек проекта: pandas, seaborn, plotly, matplotlib

********************************************************
07_tableau — проект с построением дашбордов в Tableau Public

Используя данные Яндекс.Дзена построить дашборд с метриками взаимодействия пользователей с карточками статей.

Используемый стек проекта: SQL, Python, sqlalchemy, Tableau

********************************************************

08_prefabricated_project — проект от телекоммуникационной компании

Определяем текущий уровень потребительской лояльности (NPS) среди клиентов телекоммуникационной компании. 
Строим дашборд со значеним NPS.

Используемый стек проекта: Python, Pandas, sqlalchemy, Tableau.

********************************************************

09_business_analysis — проект «Юнит-экономика»

Задача для маркетингового аналитика развлекательного приложения Procrastinate Pro+. Несмотря на огромные вложения в рекламу, 
последние несколько месяцев компания терпит убытки. Цель исследования — разобраться в причинах и помочь компании выйти в плюс.

Используемый стек проекта: Python, pandas, seaborn, matplotlib, numpy


********************************************************

10_advanced_sql — решение задач с применением продвинутых элементов языка SQL

Решаем задачи с применением продвинутых инструментов SQL

Используемый стек проекта: SQL, Оконные функции

********************************************************

11_statistical_analysis — проект «Определение перспективного тарифа для телеком-компании».

На основе данных клиентов оператора сотовой связи проанализируем поведение клиентов и поиск оптимального тарифа.

Используемый стек проекта: Python, pandas, scipy, matplotlib, math, numpy, random


********************************************************

12_AB_tests — проект «A/B-тестирование»

Используя данные интернет-магазина приоритезируем гипотезы, производим оценку результатов A/B-тестирования различными методами

Используемый стек проекта: Python, pandas, numpy, scipy.stats, matplotlib.pyplot, pandas.plotting
