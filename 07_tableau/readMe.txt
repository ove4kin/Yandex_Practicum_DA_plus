Проект «Дашборд для Яндекс.Дзен».

Цель проекта: 
Написать пайплайн, который автоматизирует процесс выгрузки сырых данных о взаимодействиях пользователей и предобработает их.
На основе данных пайплайна построить дашборд.

Используемый стек проекта: SQL, Python, sqlalchemy, Tableau
Результат проекта: Пайплайн написан, предобработка сырых данных достигнута средствами только языка SQL, дашборды построены
Статус проекта: Завершён
