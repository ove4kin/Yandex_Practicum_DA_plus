Для каждой страны отобразим общую сумму привлечённых инвестиций, которые получили компании, зарегистрированные 
в этой стране. Страну, в которой зарегистрирована компания, можно определить по коду страны. 
Отсортируем данные по убыванию суммы.

SELECT country_code,
       SUM(funding_total)
FROM company
GROUP BY country_code
ORDER BY (SUM(funding_total)) DESC;
